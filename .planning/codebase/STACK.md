# Tech Stack

## Backend
- **Framework**: FastAPI (Python 3.10+)
- **ORM / DB Toolkit**: SQLAlchemy (Async), Alembic (Migrations)
- **Validation**: Pydantic
- **Testing**: Pytest

## Data Layer
- **Relational DB**: PostgreSQL (Structured worker data)
- **Document DB**: MongoDB (Health records, dynamic symptom data)
- **Cache/Queue**: Redis

## Frontend
- **Framework**: React.js
- **Build Tool**: Vite

## AI/Models (BhashaSehat, KaamSuraksha, Document AI)
- **ASR/NLP**: Whisper, IndicBERT
- **OCR**: Tesseract / EasyOCR
- **ML**: XGBoost / LightGBM

## Infrastructure
- **Containerization**: Docker & Docker Compose
- **Static Analysis**: Pyright, Pyre-check
