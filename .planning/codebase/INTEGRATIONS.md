# Integrations Map

## Authentication & Identity
- **ABHA (Ayushman Bharat Health Account)**: National health ID integration for portability.
- **JWT**: Internal JSON Web Tokens for API authentication.

## Voice & Language Processing
- **OpenAI Whisper**: Speech-to-text (ASR) specifically tuned for Indian accents/languages.
- **IndicBERT**: Symptom extraction and NLP processing.

## Document Processing
- **Tesseract / EasyOCR**: Used for extracting text from printed and handwritten prescriptions.

## Communications
- **WhatsApp Business API**: Rich notifications and two-way worker interactions.
- **SMS Gateway**: Fallback mechanism for basic alerts.
- **IVR Provider**: Voice-based interactive alerts for low-literacy users.
